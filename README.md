# yadro_test_task

Тестовое задание для кампании Yadro  

## Description

Cтек технологий: 
- React - фронтенд
- FastAPI - бэкенд
- Nginx - веб сервер

Реализован таймер, который отображает в реальном времени количество секунд с начала эпохи.  
Соединение фронтенда с бэкендом происходит с помощью websocket соединения.

## Launch

```./build.sh```
